# Кои стойности се показват от цикъла "for"?

За всеки цикъл напишете кои стойности ще се покажат. След това сравнете с отговора.

И двете цикли `показват` същите стойности или не?

1. Постфикс формата:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. Префикс формата:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
