Отговорът: първата и третата ще се изпълняват.

Детайли:

```js run
// Изпълнява се.
// Резултатът от -1 || 0 = -1, вярна
if (-1 || 0) alert( 'първи' );

// Не се изпълнява.
// -1 && 0 = 0, не вярна
if (-1 && 0) alert( 'втори' );

// Изпълнява се.
// Операторът && има по-голям приоритет от ||
// така че -1 && 1 се изпълнява първи, давайки ни веригата:
// null || -1 && 1  ->  null || 1  ->  1
if (null || -1 && 1) alert( 'трети' );
```
