# Пренапиши конструкцията от "switch" към "if"

Напиши кода използвайки `if..else` което ще кореспондира с `switch`:

```js
switch (browser) {
  case 'Edge':
    alert( "Имаш Edge!" );
    break;

  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    alert( 'Окей, поддържаме тези браузъри също' );
    break;

  default:
    alert( 'Надяваме се тази страница да изглежда ОК!' );
}
```
